# TaskFlow – Project Management System (PMS)

## 📌 Description

TaskFlow is a Jira-like Project Management System that helps users manage tasks, projects, and workflows efficiently. It also includes AI-based features for generating documents like BRD and PRD.

---

## 🛠 Tech Stack

* Frontend: React / Svelte (update correctly)
* Backend: Node.js, Express.js
* Database: MongoDB

---

## ⚙️ Features

* Task creation, update, delete (CRUD)
* Kanban board for workflow management
* AI-based BRD & PRD generation
* Project tracking dashboard
* API-based communication

---

## 🏗 Architecture

Client-Server Architecture:

Frontend → API → Backend → Database

* Frontend handles UI and user interaction
* Backend handles business logic and APIs
* Database stores users, tasks, and projects

---

## ▶️ How to Run

### Frontend

cd frontend
npm install
npm run dev

### Backend

cd backend
npm install
npm start
